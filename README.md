# Interactive Visualization and Performance Analysis of IPL Using Power BI

## Overview
This research paper focuses on analyzing Indian Premier League (IPL) data using Power BI and Business Intelligence techniques.

## Technologies Used
- Power BI
- DAX
- Power Query
- Data Analytics
- Data Visualization

## Research Highlights
- Interactive dashboard development
- Team performance analysis
- Player statistics analysis
- Venue and match trend analysis
- KPI-driven insights
